# Images Directory

## Team Photos
Place team member photos in the `team/` subdirectory with the following naming convention:

- `john-smith.jpg` - Project Lead photo
- `sarah-johnson.jpg` - Developer photo  
- `michael-chen.jpg` - Marine Biologist photo

## Photo Requirements
- Format: JPG or PNG
- Size: Minimum 300x300 pixels (square aspect ratio preferred)
- File size: Under 1MB for optimal loading

## Fallback Display
If photos are not available, the page will automatically display fallback icons with the person's initials in a colored circle.

## Adding New Team Members
1. Add photo to `/images/team/` directory
2. Update `about.html` with new team member information
3. Follow the existing HTML structure for consistency
